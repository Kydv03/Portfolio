# Portfolio Website

This is my personal portfolio showcasing my projects, skills, and experience as a developer.  
Built using **HTML, CSS, and JavaScript**, it is fully **responsive** and includes some **scroll animations** for a modern look.

## 🛠 Features
- Responsive design for desktop and mobile
- Smooth scroll and scroll-based animations
- Sections for About Me, Skills, Projects, and Contact
- Clean UI/UX with interactive elements

## 💻 Technologies Used
- HTML
- CSS (Flexbox/Grid, animations)
- JavaScript (DOM manipulation, scroll effects)
